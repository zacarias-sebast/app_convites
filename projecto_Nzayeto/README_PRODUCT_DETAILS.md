# 🚀 Tela de Detalhes de Produto - Nzayeto

Uma implementação profissional e completa de página de detalhes de produto com:

✅ **Galeria de imagens** com carrossel  
✅ **Seleção de variantes** (cor, tamanho, preço)  
✅ **Atributos customizados** do banco de dados  
✅ **FAQ** expandível  
✅ **Integração Supabase**  
✅ **Design responsivo** (mobile, tablet, desktop)  
✅ **shadcn/ui** + **Tailwind CSS**  

---

## ⚡ Começar Agora (3 passos)

### 1. Testar SEM Supabase (Imediato)
```bash
npm run dev
# Abra: http://localhost:3000/products/demo
```

### 2. Configurar Supabase (5 minutos)
- Execute SQL de `database/product-schema.sql`
- Insira dados de `database/sample-data.sql`

### 3. Acessar sua página
```
http://localhost:3000/products/[seu-id]
```
